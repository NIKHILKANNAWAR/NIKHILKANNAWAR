## Projects

### Comparision-and-Implementation-of-Hybrid-Retriever-with-Sparse-Retriever-and-Dense-Retriever

Developed a Retrieval Augmented Generation system that answers programming questions related to Python and Java documentation.

The system processes official documentation, converts it into structured text chunks, and indexes them using both sparse retrieval (BM25) and dense retrieval (FAISS embeddings). A hybrid ranking approach using Reciprocal Rank Fusion improves retrieval quality before sending contextual information to a Gemini language model for answer generation.

The project also includes an evaluation framework measuring Precision@K, Recall@K, and Mean Reciprocal Rank across different retrieval strategies, along with a deployed Streamlit application for interactive querying.

### FMCG-Data-Consolidation-with-Databricks-Lakehouse

Built an end to end data engineering pipeline to consolidate fragmented data sources from parent and child companies in the FMCG domain.

Implemented a Lakehouse architecture using Databricks with Bronze, Silver, and Gold layers. Raw data stored in AWS S3 is ingested using PySpark, transformed into Delta tables, and processed to create dimension and fact tables.

The pipeline includes data cleaning, schema enforcement, incremental loading, and the creation of analytics ready views that support downstream reporting and dashboarding.

### Superstore_Sales_Analysis_Using_SQL

Designed a SQL based analytics workflow using the Superstore Sales dataset to simulate a real world analytical data pipeline.

The project includes raw data ingestion, validation checks, data cleaning, and transformation into a dimensional star schema with fact and dimension tables. Analytical queries were developed to generate insights such as revenue trends, customer purchasing patterns, product performance, and regional sales contributions.

The project demonstrates advanced SQL techniques including window functions, common table expressions, aggregations, and time based analysis.

### Innovative-Monitoring-System---TeleICU

Developed a monitoring system for ICU environments that detects the presence of patients, doctors, nurses, and relatives using YOLOv8 object detection.

When a patient is detected to be alone in the room, MediaPipe is used to analyze body movement and facial expressions to monitor patient activity. The system processes video input and generates movement analytics that help support continuous monitoring in critical care environments.

### Personalized Health Recommendation Based on Ayurveda Using Machine Learning

Built a machine learning based system that predicts individual Prakriti, Dosha, and Agni profiles from user inputs to provide personalized Ayurvedic health recommendations.

The system uses an ensemble stacking classifier along with GAN based data augmentation to improve model performance with limited training data. A Streamlit application was developed to provide users with personalized health guidance related to diet, lifestyle, yoga practices, rejuvenation, and remedies.
